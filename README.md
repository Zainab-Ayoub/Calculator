# Calculator

A simple web-based calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clear display functionality
- Error handling for invalid expressions

## Demo

![Calculator Demo](calculator.gif)

## How to Use

1. Open `index.html` in your web browser.
2. Click the number buttons to enter digits.
3. Click operator buttons (+, -, x, /) to select operations.
4. Click "=" to calculate the result.
5. Click "C" to clear the display.

## Files

- `index.html`: The main HTML structure
- `styles.css`: Styling for the calculator
- `calculator.js`: JavaScript logic for functionality
